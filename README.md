# 🧮 Calculator Web Application

A simple and responsive calculator web application built using HTML, CSS, and JavaScript. This calculator performs basic mathematical operations such as addition, subtraction, multiplication, and division.

---

# 🚀 Features

✅ Addition  
✅ Subtraction  
✅ Multiplication  
✅ Division  
✅ Clear Screen Functionality  
✅ Delete Last Character  
✅ Responsive User Interface  
✅ Interactive Button Design  

---

- HTML5

- JavaScript
